## 開発環境準備

```bash
# パッケージをインストール
$ pnpm i
```

## スクリプト実行方法(Website 指定)

```bash
# Type Check
$ pnpm ●●● typecheck

# Lint
$ pnpm ●●● lint:fix

# Format
$ pnpm ●●● fmt:fix

# Dev Server
$ pnpm ●●● dev

# Build
$ pnpm ●●● build

# Preview
$ pnpm ●●● start
```

## スクリプト実行方法(Project Root)

```bash
# Type Check
$ pnpm typecheck

# Lint
$ pnpm lint:fix

# Format
$ pnpm fmt:fix
```
